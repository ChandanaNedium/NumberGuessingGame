# NumberGuessingGame

The Guess the Number Game is a Java-based game where the computer generates a random number between 1 and 100. The player must guess the correct number within limited attempts.

//Features

Random number generation

Hint system (Too High / Too Low)

Multiple rounds

Score calculation

Limited attempts

//Technologies Used

Java

Random class

Scanner class

//How to Run

Compile:

javac GuessTheNumberGame.java

Run:

java GuessTheNumberGame
